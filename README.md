# STM32 AT指令解析器
基于STM32F103C8T6实现的AT指令解析器，采用状态机方式解析，支持串口收发。

## 功能
- 解析基础AT指令，返回OK；
- 解析AT+TEST指令，返回OK: TEST；
- 未知指令返回ERROR: Unknown CMD；
- 串口波特率：9600 8N1。

## 文件说明
- main.c：主函数，初始化串口和系统时钟；
- at_parser.c：核心解析逻辑（状态机+串口初始化+中断）；
- at_parser.h：头文件，定义枚举/结构体/函数声明。

## 使用方法
1. 用Keil5打开工程，勾选Use MicroLIB；
2. 编译下载到STM32F103C8T6；
3. 串口助手配置9600 8N1，发送AT指令即可收到OK。
